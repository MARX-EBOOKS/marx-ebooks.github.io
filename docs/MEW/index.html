说明

　　MARX ENGELS WERKE（MEW，直译为马克思恩格斯著作）目前共出版44卷。其中前43卷由民主德国柏林狄茨出版社出版，1-39卷篇目按照马恩全集俄文第二版编排，与马恩全集中文第一版篇目、卷次划分基本一致，40-43卷为增补卷，包含部分马克思恩格斯早期著作、政治经济学手稿，编排顺序与中文版不同，44卷及之后各卷目前由德国罗莎·卢森堡基金会续编。卢森堡基金会同时也在组织重订、再版1-43各卷，出版情况可参考狄茨出版社官方页面及有关宣传册。
　　而以上网页中的内容均来源于东德版MEW，由mlwerke.de自1998年起整理录入，非罗莎卢森堡基金会重订版，缺41、43卷内容，以及数字方括号注释。同时，因录入年代久远，网页也存在一定的兼容问题，网页样式也较为陈旧，格式也较不规整，且经多次转载（爬取）后，部分内容也出现乱码、缺失。
　　综上原因，以上网页内容仅供参考。柏林卡尔·狄茨出版社计划于2026年推出MEW1-44卷的电子版，如有需要，届时请以官方版为准。
